npm install 时提示没有C:\Users\abcd\AppData\Roaming\npm路径,将node.js安装目录下的拷贝过去

提示npm only allow pnpm，
pnpm install提示node.js下没有pnpm:
执行以下语句：npm install -g pnpm
如果计算机开启代理则要关闭后安装
然后就可以正常执行了

npm run dev 提示：Error: Cannot find module 'C:\Users\abcd\AppData\Roaming\npm\node_modules\npm\bin\npm-cli.js'
解决办法：拷贝安装目录下的npm文件夹至C:\Users\%USERNAME%\AppData\Roaming\npm\node_modules
Copy the directory named npm from your installed node path (In my case the npm directory was available in C:\Program Files\nodejs\node_modules).
Navigate to C:\Users\%USERNAME%\AppData\Roaming\npm\node_modules and paste the copied npm directory there.
